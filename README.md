Aplicación web para simular la gestión de inventario de un kiosco. Incluye agregar, editar productos y cotizar totales.

1. Abre index.html en un navegador.
2. Saluda ingresando tu nombre.
3. Agrega productos con nombre, cantidad y precio.
4. Vende/Elimina productos existentes.
5. Edita productos desde la lista.
6. Cotiza el total del inventario.
7. Ve a reportes para estadísticas.

- Inventario carga desde JSON y se guarda en localStorage.
- Operaciones validan datos y actualizan en tiempo real.
- Enfoque: Simulación de ventas sin logins ni pagos.
- HTML/CSS/JS + Toastify + SweetAlert2.

Este proyecto lo pense de tal manera que sea escalable en un futuro, agregandole secciones de cosas puntuales (Ejemplo: verduras).
